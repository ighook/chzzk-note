# Chzzk Note

치지직 라이브를 보다가 남기고 싶은 순간을 바로 기록하는 Chrome 확장 프로그램

---

방송 화면에서 `G` 키를 누르면 현재 방송 정보와 재생 시간을 담은 메모 팝업이 열립니다. 저장한 메모는 확장 프로그램 팝업에서 한 번에 확인하고 수정하거나 삭제할 수 있습니다.

## 한 줄 요약

> 방송을 끊지 않고, 지금 보고 있는 장면의 시간과 메모를 함께 남기는 치지직 전용 타임 마커입니다.

## 주요 기능

| 기능      | 설명                                                         |
| --------- | ------------------------------------------------------------ |
| 빠른 메모 | 치지직 방송 화면에서 `G` 키로 메모 팝업을 엽니다.            |
| 타임 마킹 | 채널명, 방송 경과 시간, 작성일을 메모와 함께 저장합니다.     |
| 메모 관리 | 확장 프로그램 팝업에서 저장된 메모를 확인, 수정, 삭제합니다. |
| 로컬 저장 | Chrome extension local storage에 메모를 저장합니다.          |

## 사용 흐름

1. `https://chzzk.naver.com/*` 페이지에서 방송을 시청합니다.
2. 메모를 남기고 싶은 순간에 `G` 키를 누릅니다.
3. 열린 팝업에 메모를 입력하고 저장합니다.
4. 브라우저 확장 프로그램 팝업에서 저장된 메모를 확인합니다.
